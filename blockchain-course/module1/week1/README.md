# Week 1: Blockchain Architecture & Basic Cryptography

## Topics
- What is blockchain? Distributed ledger technology
- Hash functions and their properties
- Public-key cryptography
- Digital signatures
- Merkle trees

## Learning Objectives
By the end of this week, students will be able to:
1. Explain the fundamental concepts of blockchain technology
2. Understand cryptographic hash functions and their properties
3. Implement public-key cryptography for secure communications
4. Create and verify digital signatures
5. Build Merkle tree data structures

## Theory Session (2 hours)
### 1. Introduction to Blockchain (30 min)
- Definition and core characteristics
- History and evolution
- Comparison with traditional databases

### 2. Distributed Ledger Technology (30 min)
- Decentralization concepts
- Consensus mechanisms overview
- Network topology

### 3. Cryptographic Foundations (60 min)
- Hash functions (SHA-256)
- Properties: determinism, quick computation, pre-image resistance, small changes, collision resistance
- Public-key cryptography overview
- Digital signatures
- Merkle trees structure and benefits

## Implementation Session (3 hours)
### 1. Setting Up the Development Environment (30 min)
- Go installation and setup
- Required packages and dependencies
- Project structure

### 2. Basic Block Structure Implementation (60 min)
- Creating the Block struct
- Understanding block fields
- Timestamp, data, previous hash, current hash, nonce

### 3. Cryptographic Utilities (90 min)
- SHA-256 hashing implementation
- Public-key cryptography functions
- Digital signature system
- Merkle tree implementation

## Project Work (2 hours)
### Hands-on Projects:
1. Create a simple block structure
2. Implement SHA-256 hashing
3. Build Merkle tree implementation
4. Digital signature system

## Assessment
- Code review of implemented components
- Quiz on cryptographic concepts
