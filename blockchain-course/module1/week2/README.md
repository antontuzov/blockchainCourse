# Week 2: Building Your First Blockchain

## Topics
- Block structure and chain linking
- Proof of Work consensus
- Difficulty adjustment
- Blockchain validation
- Genesis block creation

## Learning Objectives
By the end of this week, students will be able to:
1. Implement a complete blockchain structure
2. Understand and implement Proof of Work consensus mechanism
3. Adjust mining difficulty based on block times
4. Validate blockchain integrity
5. Create and manage the genesis block

## Theory Session (2 hours)
### 1. Blockchain Structure (30 min)
- Chain linking concepts
- Block references and immutability
- Genesis block significance

### 2. Consensus Mechanisms (45 min)
- Proof of Work explanation
- Mining process
- Difficulty adjustment algorithms

### 3. Blockchain Validation (45 min)
- Chain integrity checks
- Block validation rules
- Fork resolution

## Implementation Session (3 hours)
### 1. Blockchain Structure Implementation (60 min)
- Blockchain struct definition
- Adding blocks to the chain
- Chain traversal methods

### 2. Proof of Work Implementation (90 min)
- POW struct and methods
- Mining algorithm
- Difficulty adjustment

### 3. Validation System (30 min)
- Blockchain integrity verification
- Block validation functions

## Project Work (2 hours)
### Hands-on Projects:
1. Complete blockchain implementation
2. Proof of Work miner
3. Blockchain validation system
4. CLI interface for blockchain interaction

## Assessment
- Code review of implemented blockchain
- Performance testing of mining algorithm
- Quiz on consensus mechanisms
