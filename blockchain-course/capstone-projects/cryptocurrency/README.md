# Cryptocurrency Capstone Project

## Overview
This project requires students to implement a complete cryptocurrency system from scratch, incorporating all the concepts learned throughout the course.

## Features to Implement
1. Blockchain data structure
2. Proof of Work consensus
3. Wallet system with public/private key cryptography
4. Transaction system with UTXO model
5. P2P network with multiple nodes
6. CLI and API interfaces

## Implementation Guidelines
- Use Go as the primary implementation language
- Follow secure coding practices
- Include comprehensive testing
- Provide clear documentation
- Ensure proper error handling

## Directory Structure
```
cryptocurrency/
├── main.go          # Entry point
├── blockchain/      # Blockchain implementation
├── wallet/          # Wallet system
├── transaction/     # Transaction system
├── network/         # P2P network implementation
├── cli/             # Command line interface
├── api/             # REST API
└── tests/           # Test suite
```

## Getting Started
1. Initialize the project structure
2. Implement the core blockchain
3. Add wallet functionality
4. Create transaction system
5. Build P2P network
6. Develop interfaces
7. Test and document
