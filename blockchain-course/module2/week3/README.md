# Week 3: Transactions & UTXO Model

## Topics
- Transaction structure
- Unspent Transaction Output (UTXO) model
- Scripting system basics
- Transaction validation
- Wallet addresses

## Learning Objectives
By the end of this week, students will be able to:
1. Implement a complete transaction structure
2. Understand and implement the UTXO model
3. Create and validate transaction scripts
4. Implement transaction validation mechanisms
5. Generate and manage wallet addresses

## Theory Session (2 hours)
### 1. Transaction Structure (30 min)
- Transaction components: inputs and outputs
- Transaction ID generation
- Serialization and deserialization

### 2. UTXO Model (45 min)
- Unspent Transaction Outputs explanation
- UTXO set management
- Balance calculation using UTXO

### 3. Scripting System (45 min)
- Script basics and opcodes
- Locking and unlocking scripts
- Script validation

## Implementation Session (3 hours)
### 1. Transaction Structure Implementation (60 min)
- Transaction struct definition
- TXInput and TXOutput structs
- Transaction ID generation

### 2. UTXO Model Implementation (90 min)
- UTXO set management
- Balance calculation
- UTXO selection algorithms

### 3. Scripting System (30 min)
- Script execution engine
- Basic opcodes implementation
- Script validation

## Project Work (2 hours)
### Hands-on Projects:
1. UTXO-based transaction system
2. Transaction signing and verification
3. Basic wallet implementation
4. Balance calculation

## Assessment
- Code review of implemented transaction system
- UTXO set management testing
- Quiz on transaction validation
