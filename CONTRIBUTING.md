# Contributing to Blockchain Development Course

Thank you for your interest in contributing to the Blockchain Development Course by Anton Tuzov! This document provides guidelines for contributing to this educational repository.

## How to Contribute

### Reporting Issues
If you find any issues with the course materials, code examples, or documentation:
1. Check if the issue has already been reported
2. If not, create a new issue with a clear title and detailed description
3. Include steps to reproduce the issue if applicable
4. Add any relevant screenshots or error messages

### Suggesting Enhancements
We welcome suggestions for improving the course:
1. Open an issue describing your proposed enhancement
2. Explain why the enhancement would be valuable
3. If possible, provide examples or mockups

### Code Contributions
To contribute code fixes or improvements:

1. Fork the repository
2. Create a new branch for your changes
3. Make your changes with clear, concise commit messages
4. Ensure all tests pass
5. Submit a pull request with a detailed description of your changes

### Documentation Improvements
Help improve the course documentation by:
- Fixing typos or grammatical errors
- Clarifying confusing sections
- Adding examples or diagrams
- Updating outdated information

## Style Guidelines

### Code Style
- Follow Go's official style guide
- Use meaningful variable and function names
- Add comments for complex logic
- Write clear, concise documentation

### Documentation Style
- Use clear, simple language
- Maintain consistent formatting
- Include code examples where helpful
- Proofread for grammar and spelling

## Testing
Before submitting any code changes:
1. Run all existing tests to ensure nothing is broken
2. Add new tests for any new functionality
3. Ensure test coverage is maintained or improved

```bash
go test ./...
```

## Questions?
If you have any questions about contributing, feel free to open an issue or contact the course author directly.

Thank you for helping make this blockchain development course better for everyone!
